## Build Setup

```bash
# 克隆项目
git clone

# 进入项目目录
cd dataManage

# 安装依赖
npm install

# 可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npmmirror.com

# 启动服务
npm run dev
```

浏览器访问 [http://localhost:4404](http://localhost:4404)

## 发布

```bash

# 构建生产环境
npm run build
```

## 其它

```bash
# 预览发布环境效果
npm run preview

# 代码格式检查并自动修复
npm run lint -- --fix
```
